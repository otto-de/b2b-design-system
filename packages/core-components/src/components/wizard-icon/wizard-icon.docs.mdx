# Wizard Icon Component

The wizard icon is a wrapper of the [Rounded Icon](?path=/docs/components-assets-rounded-icon--story-010-icon) that encapsulates the styles to expose an API with the states
of the icon in the context of the wizard.

<br />

## Properties

### state

Currently, we support this states states: `default`, `completed`, `pending` and `disabled`

<Story id="components-status-feedback-wizard--story-020-wizard-states" />

### step

From 1 to 6, the step number of the icon

### checkIcon

Defaults to true. It will show a checkmark icon when a step is completed.
If set to false, it will show the step number.
