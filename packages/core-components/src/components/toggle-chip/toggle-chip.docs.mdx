# Toggle Chip

<Canvas columns={3} withSource="open" withToolbar={false}>

The Toggle Chip component is like a button, allowing users to toggle between active
and inactive states.

<br />

## Properties

### Label

The visible text displayed inside the chip.

### Name

The identifier for the input element, useful for form submission or accessibility.

### Active

A boolean state indicating whether the chip is currently selected. Default is set to false.

### Disabled

A boolean state that indicates whether the chip is clickable. Default is set to false.

## Events

### b2b-selected

This event is emitted with current value whenever the toggle chip is selected.

```typescript
export interface ToggleChipEventDetail<T = string> {
  value: T;
}
```
