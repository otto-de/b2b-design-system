# Scrollable Container

This component is a helper container that encapsulates the classes to make the content scrollable both horizontally
and vertically. The scroll will only appear when the content is bigger than the container.

This container sets the `white-space` property as `nowrap`, so be cautious to overwrite it in any children if you
require the text wo wrap.

```html
<b2b-scrollable-container style="width: 400px; height: 50px">
  <!-- Any content -->
</b2b-scrollable-container>
```
