# Anchor Component

The anchor pattern provides the visual formatting of links, which also includes the hover effect with adjustment of the mouse pointer.
The default anchor is underlined and turns red when being hovered. The font size and text color depend on the parent element and can vary according to the basic colors.

## Default Arguments

<Canvas columns={3} withSource="open" withToolbar={false}>
  <Story id="components-interaction-anchor--story-010-default" />
</Canvas>

The only required argument to the link component is the `href` attribute, all other arguments are optional.
If not specified, `download` is set to null, so that a default anchor is not downloadable.
The default `target` attribute is `self`, the default `size` is inherited from the anchor's parent element.

## Properties

### Size

<Canvas columns={3} withSource="open" withToolbar={false}>
  <Story id="components-interaction-anchor--story-020-size-100" />
</Canvas>

If not specified, the size of the anchor component is inherited from the parent element.
Alternatively, it can be set to `size="100"` which defines the font size as 14px.

### Hover Color

<Canvas columns={3} withSource="open" withToolbar={false}>
  <Story id="components-interaction-anchor--story-040-blue-hover-color" />
</Canvas>

If not specified, the hover-color of the anchor component is set to default as b2b-color-red-100.
Alternatively, it can be set to `hover-color="inherit"` which inherits the color from parent element.

### Target

<Canvas columns={3} withSource="open" withToolbar={false}>
  <Story id="components-interaction-anchor--story-030-target-blank" />
</Canvas>

The default target of the anchor component is self, which will open the target in the same window and add another entry to the navigation stack.
If specified to blank, the target will open in another tab instead.

### IFrame Support

The anchor component also supports two other `target` types - parent and top, that are exclusively used in the context of IFrames.
If specified to `parent` the target will open in the IFrame that it is used in.

If specified to `top`, the target will open in the IFrame that is closest to the document root.
