# Toggle Switch

The toggle switch component used for the switch. Additionally, a label can be included right or left of the toggle switch.

<Canvas of={ToggleSwitchStories.SwitchOn} sourceState="shown" />

## Properties

### Label

The label of the toggle button. This is required

### Label Position

This property controls whether the label should be rendered on the right or left side of the toggle switch. Its value should be either `left` or `right`. It is optional and defaults to `left`.
