# Background Box Component

<Canvas of={BackgroundBoxStories.Default} />

The BackgroundBox is a utility component designed to serve as a container for other components, featuring a white background and a border.

The BackgroundBox component adjusts its width (maxWidth), padding (noPadding), and border visibility (borderTop, borderRight, borderBottom, borderLeft). The height depends on its internal content.

NOTE: All properties are optional. The component can simply be used like this without specifying any properties aside from the label:

<Source
  code={`<b2b-background-box><b2b-headline>Background Box</b2b-headline></b2b-background-box>`}
/>

## Properties

### maxWidth

The Background Box component can be configured to have maxWidth. When maxWidth is true, it fills the available width until 1212px.

<Canvas of={BackgroundBoxStories.MaxWidth} />

### noPadding

The noPadding property controls internal spacing. If true, no padding is applied; if false, a default padding of 24px is added.

<Canvas of={BackgroundBoxStories.NoPadding} />

### Border Control

Customize border visibility with border-\*properties (top, right, bottom, left). Set any border to "none" to hide it.
